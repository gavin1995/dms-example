## DMS示例项目

### 快速开始

**安装**

```bash
> git clone https://github.com/gavin1995/dms-example.git
> cd [with-*]
> yarn # 若没有yarn，请使用 npm install
```

**按需修改配置**

* 将项目根目录下的`.config.js`改名为`config.js`
* 对`config.js`按需进行配置修改

**启动示例项目**

```bash
> cd [with-*] # 进入示例项目
> yarn start # 启动，若没有yarn，请使用 npm run start
```
